## ブランチ運用方針
- master  :正式版
- develop :開発用
- test    :検証・立ち上げ期などの不安定なコード（安定化したら消す）

## 初期方針
- 先ずはtestブランチで遊ぶ
- testブランチからさらに子ブランチを生やして遊ぶとかは自由にどうぞ
- developとmasterは一旦はいじらない
- 立ち上げ期が終わったらtestブランチの内容をmasterに（手動）マージ
